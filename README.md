# IDE 1.0 - Panel de Tasas

Aplicación web ligera para la gestión y conversión de tasas cambiarias (USD/EUR/USDT).

## Características
- Conversión en tiempo real.
- Modo de tasas programadas.
- Integración con APIs externas (DolarAPI, CriptoYa).
- Soporte PWA (Offline).

## Stack Técnico
- HTML5
- TailwindCSS
- JavaScript (Fetch API)

## Notas para el Desarrollador
- Mantener las rutas de las APIs actualizadas.
- Revisar la lógica de los event listeners en los inputs para evitar conflictos con el renderizado.
